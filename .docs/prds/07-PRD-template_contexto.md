# TAREFA: Biblioteca de Templates de Stack

**Funcionalidades:**
1. Ao criar um projeto, oferecer dropdown "Carregar Template".
2. Opções hardcoded no frontend (ex: "Next.js + Supabase", "Python + FastAPI", "MERN Stack").
3. Ao selecionar, preencher automaticamente os campos "Tech Stack" e "Regras de Código".

**Frontend Requirements:**
- Arquivo de configuração `templates.ts` contendo objetos JSON com os presets.
- Botão "Aplicar Template" no form de criação de projeto.
